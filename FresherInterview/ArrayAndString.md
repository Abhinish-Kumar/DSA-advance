## Arrays and Strings

| Problem                                                    | Description                                                                 |
|------------------------------------------------------------|-----------------------------------------------------------------------------|
| **Find the largest/smallest element in an array**          | Traverse the array to find the maximum or minimum value.                    |
| **Reverse an array or string**                             | Reverse the elements of an array or characters of a string.                 |
| **Check if an array is a palindrome**                      | Determine if the array reads the same forward and backward.                 |
| **Two Sum Problem**                                        | Find pairs in an array that sum to a given target.                          |
| **Move all zeroes to the end of an array**                 | Rearrange the array so that all zeroes are at the end, maintaining order.   |
| **Find the length of the longest substring without repeating characters** | Identify the longest substring in a string with all unique characters. |
